Dynamic Content Information Management System for Las Piñas City

Project information

  • Project Title: Dynamic Content Information Management System for Las Piñas City
  • Languages used: HTML, CSS, JavaScript, PHP, MySQL
  • Website URL: https://thecityoflaspinas.ml/
  • GitHub URL: https://github.com/Tiansing/Interactive-Information-Management-System-of-Las-Pinas-City.git

The Dynamic Content Information Management System developed for Las Piñas City represents an enhanced version of the existing municipality website. This improved iteration aims to provide a modernized and engaging experience for web visitors by incorporating core features such as Virtual Tour, News Room, and an Interactive Chat System.
