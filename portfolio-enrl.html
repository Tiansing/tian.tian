Web based Enrollment Management System

Project information

  • Project Title: Online Enrollment Management System
  • Languages used: HTML, CSS, JavaScript, PHP, MySQL
  • Website URL: https://tiansing.github.io/Online_Enrollment_System/
  • GitHub URL: https://github.com/Tiansing/Online_Enrollment_System

The Online Enrollment Management System is a Web-based system and it was developed for the particular department to eliminate and supersede the usage of traditional methods of enrolling students. This website facilitates the process of enrollment and management of student information.
