WeResponse Community Website

Project information

  • Project Title: WeResponse Community Website
  • Languages used: HTML, CSS, JavaScript, PHP, Firebase
  • Website URL: https://github.com/weresponse/weresponse.github.io
  • GitHub URL: https://weresponse.github.io/index.html

The WeResponse website aims to become the most reliable medium of emergency alerts, if you contact WeResponse, it will call all the possible emergency aid you needed. WeResponse have all the emergency aid that people usually call, there is no need to call every emergency aid one by one, WeResponse will call every emergency aid you needed for faster response to the emergency situation.
