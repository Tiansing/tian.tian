Math Game developed using Java

Project information

  • Project Title: Math Game
  • Language used: Java
  • GitHub URL: https://github.com/Tiansing/3dMathGameJava

This Math Game is an interactive system developed using Java. The game lets users practice and challenge their mathematical problem-solving skills while fighting against different enemies. The users' experience while playing this game is like playing an arcade because of the 3D characters.
